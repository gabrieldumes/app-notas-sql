## Funcionalidades:

* Criar, editar e ver notas
* Arquivar, desarquiviar, remover e restaurar notas
* Aplicar filtros e pesquisar notas
* Ver estatísticas
* Alterar configurações como tamanho de fonte e cor padrão das notas
